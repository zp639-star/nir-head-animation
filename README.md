# Project Page Setup

This folder is a plain static site for GitHub Pages.

## Edit content

- Main page structure: `docs/index.html`
- Styles: `docs/assets/styles.css`
- Links, BibTeX, and media slots: `docs/assets/main.js`

## Add media

- Put images in `docs/assets/media/images/`
- Put videos in `docs/assets/media/videos/`
- Put the supplemental PDF in `docs/assets/media/pdfs/`
- Then update the matching `src` or link fields in `docs/assets/main.js`

## Deploy with GitHub Pages

On GitHub, open:

`Settings -> Pages -> Build and deployment -> Deploy from a branch -> main -> /docs`

Then your site will publish from this folder.
